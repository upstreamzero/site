# Visual Direction

Don't copy any company.

Study them.

Intercom
- Clean
- Editorial
- Spacious
- Product feels alive

Stripe
- Sophisticated
- Strong typography
- Premium

Linear
- Extremely polished
- Great interaction
- Calm

Nature
- Feels like a research institution

GitHub
- Everything links.
- Provenance.
- Objects connected.

Wikipedia
- Dense knowledge graph.

Apple
- Confident.
- Lots of whitespace.
- Lets content breathe.

Overall feeling

Research institution meets modern software company.

Not startup.

Not agency.

Not academic.

Timeless.

Quiet confidence.

Evidence first.

Every page should feel inspectable.

Everything should feel connected.
